# 七秒讀書會 · 7 Second Book Club

A personal reading notes site. Books I've read, ideas I don't want to forget.

🌐 **Live site:** https://kassielooo.github.io/7secondbookclub

---

## Structure

```
├── index.html         ← main page
├── about/             ← about page
├── css/               ← shared styles
├── logo.png
└── slides/
    ├── SlideFollow.png
    ├── book01/        ← Bk01_*.png
    ├── book02/
    └── ...
```

## Adding a new book

1. Upload PNGs into `slides/book0X/`
2. Update `index.html` — add to `BOOK_NAMES`, `BOOKS`, and the HTML grid
3. Commit — site updates within ~1 minute
